WebOS Seed app
==============

## Requirements
- [WebOS Sdk](http://webostv.developer.lge.com/sdk/install-instructions/installing-sdk/)

## Commands

Open emulator on MacOS
```bash
gulp emulator[macos]
```

Launch app
```bash
gulp run-app
```
